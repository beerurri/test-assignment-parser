# WB Parser

Парсер каталога товаров Wildberries с экспортом в Excel.

## Что делает

- Выполняет поиск товаров по заданному запросу (QUERY в `config/settings.py`)
- Собирает подробные карточки товаров с параллельными запросами
- Экспортирует результаты в Excel с возможностью фильтрации по рейтингу, цене и стране производства

## Зависимости

```bash
pip install requests pandas pydantic openpyxl
```

## Использование

```bash
python main.py
```

Результаты сохраняются в файлы `catalog.xlsx` и `filtered.xlsx` в текущей директории.
